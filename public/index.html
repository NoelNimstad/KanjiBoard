The Kanji Board library is very small and easily embeded into your own projects. All documentation needed can be found on this page.

Information / Credits

Kanji Board is a small library designed to be used in Japanese learning applications to practise handwriting. It matches the strokes drawn by the users to path data from kanji SVGs, which is pulled from the KanjiVG project. The project was created as an offshoot of a Kanji-handwriting learning application I previously developed, now focused on improving the reusability of the canvas element, which I struggled to find a good solution for.
