Setup

To add KanjiBoard to your own webpage, first add the bundeled script. This can either be done through a local copy or by linking to the raw GitHub® distribution at https://raw.githubusercontent.com/NoelNimstad/KanjiBoard/refs/heads/master/build/dist/kb.bundle.js. The loading does not have to be deferred, nor does it have to be typed as a module.
